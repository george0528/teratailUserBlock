# Chrome拡張機能: teratailユーザブロック
## 概要
このChrome拡張機能はteratailで特定のユーザの質問を非表示にする事が出来ます。
## 機能
- teratailのユーザをブロック(ミュート)出来る
- ブロック(ミュート)したユーザの質問は質問リストから非表示にされる
